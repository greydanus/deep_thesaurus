The Deep Thesaurus

By the Word Wumbo Team: Eric Frank, Xinyu Hu, and Sam Greydanus


Overview

In the sentences "You are pretty great" and "You are pretty and great," the word pretty carries different meanings. Yet most standard thesauruses neglect to model this distinction, recommending a set of synonyms for the word regardless of context. This is a poor idea. It’s also penniless. Given modern language models, we can make substantial improvements over existing thesaurus APIs by taking sentence-level context into account. In particular, this can be done by using a natural language deep learning model (e.g. BERT [1], CTRL [2], or GPT-3 [3]) to rank synonyms based on their context.


How it works

Our deep thesaurus ranks synonyms using an API call to a pretrained deep learning model. We use the model to encode entire sentences as vectors and then perform arithmetic on them in order to see how a given synonym changes the meaning of the sentence. The goal is to obtain synonyms for the word “current.” This is a challenging task because the word’s meaning differs according to the sentence it is being used in. Starting from an unordered set of 36 synonyms -- some meant to replace “current” as in “contemporary” and others meant to replace “current” as in “the flow of something” -- we substitute “current” for each synonym and then use BERT to encode the resulting sentences. Then we use the cosine similarity between those sentence vectors and the original sentence vector to rank the synonyms. Intuitively, this ranking picks out the synonyms that change the meaning of the sentence the least, which are the synonyms that are most relevant to the user.

In the future, we'd like to expand our demo’s capabilities to include 1) grammar checking, 2) summarization and re-phrasing and 3) text style transfer. Our advantage over other tools, like Grammarly, is that we aim to build an elegant front-end API that can leverage whatever deep learning model is currently state-of-the-art in language modeling. If (and when) a much better NLP model is released, our writing tool will immediately improve in proportion to the quality of that model. So, in offloading the nuances of word sense disambiguation, grammar, and text summarization to research-grade text models, we can take advantage of the rapid pace of improvement of the field. One additional benefit is that these models are available in other languages, so adding multilingual support is as easy as swapping out pretrained models.
