How it works

Our deep thesaurus ranks synonyms using an API call to a pretrained deep learning model. We use the model to encode entire sentences as vectors and then perform arithmetic on them in order to see how a given synonym changes the meaning of the sentence. The figure below gives an example and some details:

Figure 1: We use our preliminary model to rank synonyms based on context. The goal is to obtain synonyms for the word “current.” This is a challenging task because the word’s meaning differs according to the sentence it is being used in. Starting from an unordered set of 36 synonyms -- some meant to replace “current” as in “contemporary” and others meant to replace “current” as in “the flow of something” -- we substitute “current” for each synonym and then use BERT to encode the resulting sentences. Then we use the cosine similarity between those sentence vectors and the original sentence vector to rank the synonyms. Intuitively, this ranking picks out the synonyms that change the meaning of the sentence the least, which are the synonyms that are most relevant to the user.

In the future, we'd like to expand our demo’s capabilities to include grammar checking, re-phrasing and text style transfer. Our advantage over other tools, like Grammarly, is that we aim to build an elegant front-end API that can leverage whatever deep learning model is currently state-of-the-art in language modeling. Whenever a better NLP model is released, our writing tool immediately improves in proportion to the quality of that model. So, in offloading the nuances of word sense disambiguation, grammar, and text summarization to research-grade text models, we can take advantage of the rapid pace of improvement of the field. One additional benefit is that these models are available in other languages, so adding multilingual support is as easy as swapping out pretrained models.
